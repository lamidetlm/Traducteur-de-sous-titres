# Traducteur de Sous-titres

Une application web permettant de traduire facilement des fichiers de sous-titres en utilisant l'API DeepL.

## Fonctionnalités

- Interface glisser-déposer pour les fichiers de sous-titres
- Support des formats .srt, .vtt, et .sub
- Traduction automatique via l'API DeepL
- Support de multiples langues
- Interface utilisateur intuitive et responsive
- Téléchargement direct des sous-titres traduits

## Installation

1. Cloner le repository :
```bash
git clone [URL_DU_REPO]
cd traducteur
```

2. Installer les dépendances :
```bash
pip install -r requirements.txt
```

3. Lancer l'application :
```bash
python app.py
```

## Utilisation

1. Accédez à l'application via `http://localhost:5000`
2. Sélectionnez la langue cible dans le menu déroulant
3. Glissez-déposez votre fichier de sous-titres dans la zone prévue
4. Cliquez sur "Traduire"
5. Téléchargez le fichier traduit

## Technologies utilisées
